Research Notes:
  • usp=sharing: Traditional Google sharing parameter
  • #action=share: URL fragment approach
  • ?sharing=true: Alternative boolean parameter
  • ?usp=share: Shortened version of usp=sharing
  • ?authuser=0&usp=sharing: With explicit user context
  • ?sharedialog=true: Direct dialog parameter
  • ?dialog=share: Generic dialog parameter
